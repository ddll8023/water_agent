# 水库水质智能监测分析平台 项目结构文档

## 技术栈

| 类别 | 技术 | 说明 |
|------|------|------|
| 后端框架 | FastAPI / Uvicorn | 异步 REST API 服务，端口 3443 |
| ORM | SQLAlchemy 2.0 + aiomysql | 异步连接池管理 user/role 数据模型 |
| 认证安全 | JWT + python-jose + bcrypt | Bearer Token 签发/验证，密码哈希，RBAC 装饰器 |
| 配置管理 | Pydantic Settings | 从 .env 加载全局配置，提供单例配置对象 |
| 前端框架 | Vue 3.5 + Composition API | SPA 应用，script setup 语法 |
| UI 组件库 | Element Plus 2.9 | 表单/按钮/输入框/表格等，中文语言包 |
| CSS 方案 | Tailwind CSS 3.4 + PostCSS | 原子化 CSS，teal 主题色扩展 |
| 状态管理 | Pinia 2.3 | 认证状态与 Token 持久化 |
| 路由 | Vue Router 4.5 | 路由懒加载，路由守卫鉴权 |
| HTTP 请求 | Axios 1.7 | 自动 Token 注入，统一响应拦截 |
| 构建工具 | Vite 6.0 | 开发代理 /api → localhost:3443，端口 3444 |
| 图标库 | @element-plus/icons-vue | 全局注册所有图标组件 |
| 数据库 | MySQL 8.0 | 用户/角色业务数据存储 |
| 缓存 | Redis 7.x（aioredis） | 已初始化，未接入业务 |
| 图数据库 | Neo4j 5.x | 已初始化，未接入业务 |
| 向量数据库 | Chroma | 本地持久化目录已配置，未接入业务 |
| AI 编排 | LangChain + LangGraph | 依赖已安装，规划中 |
| 大模型 | DeepSeek-V4 / Qwen-Max | API Key 已配置，规划中 |
| 运行环境 | Python 3.12 | 项目要求最低版本 |
| 包管理 | uv（Python）/ npm（Node） | 后端使用 uv，前端使用 npm |

## 项目结构

```
水库智能监测Agent/
├── backend/
│   ├── .env                        # 环境变量：MySQL/Redis/Neo4j/JWT/LLM 配置
│   ├── .python-version             # Python 3.12 版本声明
│   ├── pyproject.toml              # 项目元数据（name/version 占位）
│   ├── requirements.txt            # 19 项依赖清单
│   ├── uv.lock                     # uv 依赖锁定文件（不展开）
│   └── app/
│       ├── main.py                 # FastAPI 入口：CORS、路由注册、启动建表
│       ├── core/
│       │   ├── config.py           # 全局配置类：数据库/缓存/图库/JWT/LLM 参数
│       │   ├── database.py         # MySQL 异步引擎与会话工厂
│       │   ├── neo4j.py            # Neo4j 图数据库驱动初始化
│       │   ├── redis.py            # Redis 异步客户端初始化
│       │   └── security.py         # JWT 签发/验证、当前用户、角色权限
│       ├── models/
│       │   ├── user.py             # 用户 ORM：字段/索引/外键->role
│       │   └── role.py             # 角色 ORM：字段/权限 JSON/唯一索引
│       ├── schemas/
│       │   ├── common.py           # ApiResponse/Pagination/ErrorCode 枚举
│       │   ├── response.py         # success()/error() 快捷响应构造函数
│       │   ├── roles.py            # 角色请求/响应 Pydantic 模型
│       │   └── users.py            # 用户 CRUD 请求/响应 Pydantic 模型
│       ├── routers/
│       │   ├── auth.py             # 认证路由：login/register/logout/me
│       │   ├── roles.py            # 角色管理路由：get_role_list
│   │   └── users.py            # 用户管理路由：列表/详情/新增/更新/重置密码
│       ├── services/
│       │   ├── auth.py             # 认证服务：注册/登录校验/JWT 签发
│       │   ├── roles.py            # 角色服务：角色列表查询
│       │   └── users.py            # 用户服务：列表/新增/详情/更新
│       └── utils/
│           ├── exception.py        # ServiceException 业务异常类
│           ├── file.py             # 文件保存与 JSON 序列化工具
│           └── logger_config.py    # 带格式化输出的日志配置器
├── frontend/
│   ├── index.html                  # HTML 入口：标题/挂载点/JS 入口
│   ├── package.json                # 项目元数据：Vue/EPlus/Pinia/Axios/Tailwind
│   ├── package-lock.json           # npm 依赖锁定文件（不展开）
│   ├── vite.config.js              # Vite 配置：端口 3444、API 代理、@ 别名
│   ├── tailwind.config.js          # Tailwind 主题：teal 色系、Inter 字体
│   ├── postcss.config.js           # PostCSS 插件：tailwindcss + autoprefixer
│   └── src/
│       ├── main.js                 # Vue 应用入口：注册 Pinia/Router/EPlus/图标
│       ├── App.vue                 # 根组件：仅 router-view 路由出口
│       ├── api/
│       │   ├── auth.js             # 认证 API：login()/register()
│       │   ├── role.js             # 角色 API：getRoleList()
│       │   └── user.js             # 用户 API：getUserList()
│       ├── assets/
│       │   └── styles/
│       │       └── main.css        # 全局样式：Tailwind 指令 + 登录页渐变
│       ├── router/
│       │   └── index.js            # 路由表（MainLayout 包裹已认证路由）+ 路由守卫
│       ├── stores/
│       │   └── auth.js             # Pinia Store：Token/userInfo/logout
│       ├── utils/
│       │   └── request.js          # Axios 封装：Token 注入 + 响应拦截
│       └── views/
│           ├── auth/
│           │   ├── Login.vue       # 登录页：毛玻璃卡片、表单校验、错误分类
│           │   └── Register.vue    # 注册页：密码确认、手机号校验、3 秒跳转
│           ├── dashboard/
│           │   └── index.vue       # 仪表盘占位页
│           ├── layout/
│           │   └── MainLayout.vue  # 主布局：侧边栏 + 顶部栏 + 内容区
│           └── system/
│               └── UserManage.vue  # 用户管理页：用户列表（已对接）+ 角色管理（占位）
├── doc/
│   ├── 模块说明文档.md              # 已实现模块的业务功能说明文档
│   ├── 水库水质智能监测分析平台_开发模块与接口设计.md  # 10 个模块接口定义
│   └── 水库水质智能监测分析平台_开发设计文档.md       # 技术架构与设计文档
├── 规范文档/
│   ├── 前端规范文档.md              # Vue 3 前端编码规范（1376 行）
│   └── 后端规范文档.md              # 后端编码规范索引（由 backend-spec skill 承载）
├── start_backend.bat               # 后端启动：uv run uvicorn app.main:app --port 3443
├── start_frontend.bat              # 前端启动：npm run dev（端口 3444）
└── .gitignore                      # Git 忽略：pycache/node_modules/.venv/.env 等
```

| `backend/.env` | MySQL/Redis/Neo4j/Chroma/JWT/DeepSeek/Qwen 连接配置 |
| `frontend/package.json` | Vue 3.5、Element Plus 2.9、Pinia 2.3、Axios 1.7、Tailwind 3.4、Vite 6.0 |

| `frontend/tailwind.config.js` | 扫描范围、teal 主题色扩展、Inter 字体 |
| `frontend/postcss.config.js` | tailwindcss + autoprefixer 插件 |
| `backend/.python-version` | Python 3.12 |
